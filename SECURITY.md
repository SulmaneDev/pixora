# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v3.0    | :white_check_mark: |
| < v3.0  | :x:                |

## Reporting a Vulnerability

We take the security of Pixora seriously. If you find a vulnerability, please report it to us as soon as possible.

1. **Do not open a public issue.**
2. Send an email to **security@pixora.ai** (or support@pixora.ai if not available).
3. Include as much detail as possible, including steps to reproduce and any potential impact.
4. Give us reasonable time to respond and fix the issue before sharing it publicly.

We appreciate our community's help in keeping Pixora secure!
